== Description ==

InHouse Tutorials RSS Dashboard Widget was published so we could provide quick tips for sites we build. The plugin adds a dashboard widget  which is a simple RSS feed of a series of helpful tutorials.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! You now have an RSS feed of InHouse tutorials on your dashboard.
4. Use 'Screen Options' on the dashboard to control which widgets display.

== Frequently Asked Questions ==
